INSTRUÇÕES

1 - Faça o download dos arquivos através do link fornecido. Verifique se o download completou-se com o tamanho total, conforme indicado.

2 - Extraia os arquivos do arquivo .zip usando programas como o WinZip ou WinRar. Aperte com o botão direito sobre o arquivo .zip e selecione a opção "Extrair". Uma pasta será criada contendo todos os arquivos do seu Tour Virtual 360°.

3 - Para inserir o Tour Virtual 360° em seu site, copie todos os arquivos para uma pasta em seu FTP, e crie um link ou um botão em seu site que chame o arquivo "index.html".

4 - Para gravar em CD/DVD, basta copiar todos estes arquivos extraídos do ZIP para o CD/DVD, incluindo o arquivo "autorun". Lembrando que o Tour Virtual exige, obrigatoriamente, um computador para ser visualizado.

5 - Para uso off-line, basta copiar todos os arquivos para um computador e abrir o arquivo "tour_testingserver" (há duas versões, uma para Mac e uma para Windows). A maioria dos navegadores possuem restrições ao abrir arquivos off-line por motivos de segurança, portanto não abra diretamente o arquivo "index.html". Não é necessário fazer o upload desse arquivo no servidor.

6 - O Tour Virtual 360° que desenvolvemos pode ser acessado online em dispositivos móveis iOS(iPod Touch, iPhone e iPad), Android (a partir da versão 2.2 do Android com Flash Player instalado ou a partir da versão 4.0 em navegador compatível com HTML5) e Windows Phone 8.
